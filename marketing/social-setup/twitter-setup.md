# Twitter/X Setup Guide for TypeSmart

## Account Creation Steps

### Step 1: Sign Up
1. Go to **twitter.com** or open Twitter app
2. Click **"Create account"**
3. Use this info:
   - **Name:** TypeSmart
   - **Phone:** Your phone number (for verification)
   - **Birthday:** Any date (doesn't matter for brands)
4. Choose **@TypeSmartAI** as handle
   - If taken, try: @TypeSmartApp, @UseTypeSmart, @TypeSmart_io
5. Complete email verification

### Step 2: Profile Setup

**Profile Picture:**
- Upload: Use TypeSmart logo or create simple text logo
- Size: 400x400px
- Style: Dark background with "TS" or sparkle icon

**Header/Banner Image:**
- Size: 1500x500px
- Design options:
  - Screenshot of TypeSmart interface
  - Dark gradient with tagline
  - Abstract AI/writing themed image

**Bio (160 characters max):**
```
AI writing assistant with perfect tone control. LinkedIn posts, emails, dating messages & complaints. Free tier: 5/day. Pro: $9/mo
```

**Location:** 
```
🌐 typesmart.io
```

**Website:**
```
https://typesmart.io
```

**Birth Date:** Skip or set to company founding date

### Step 3: Pin a Welcome Tweet

After posting your first thread, pin it to your profile.
